# Custom GPT agents (source layout)

Each subdirectory is one agent: **`prompt.md`**, **`agent.yaml`**, **`knowledge/`**, optional **`apps/`**.

| Folder | Description |
|--------|-------------|
| **`sample-agent/`** | Minimal template to copy when creating a new GPT |
| **`autospec-evaluator/`** | AutoSpec PRD→ERD evaluation workflow |

Copy `sample-agent` to a new slug, then replace prompts and knowledge.
